# Components Directory

This directory contains reusable React components for the eCommerce application.

## Structure Guidelines

- Use PascalCase for component names
- Each component should have its own folder with:
  - `ComponentName.jsx` - Main component file
  - `ComponentName.css` - Component styles (optional)
  - `index.js` - Export file for cleaner imports

## Example Structure:
```
components/
├── Header/
│   ├── Header.jsx
│   ├── Header.css
│   └── index.js
├── ProductCard/
│   ├── ProductCard.jsx
│   ├── ProductCard.css
│   └── index.js
└── Footer/
    ├── Footer.jsx
    ├── Footer.css
    └── index.js
```
