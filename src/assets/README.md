# Assets Directory

This directory contains static assets for the eCommerce application.

## Structure:
```
assets/
├── images/
│   ├── products/
│   ├── icons/
│   └── banners/
├── fonts/
└── data/
    └── products.json
```

## Guidelines:
- Keep images organized by category
- Use web-optimized formats (WebP, AVIF when possible)
- Include product images, icons, and banner images
- Store mock data files here during development
